# BlueBuket 중고마켓 프로젝트

## 프로젝트 설명

각 지역에 오프라인 센터 및 관리자가 존재하는 판매자 - 관리자 - 구매자 형태의 중고거래 사이트
판매자는 자주 진상에 시달릴 필요 없으며, 오프라인 센터가 별도 존재하므로 구매자도 직접 상품을 보고 결정할 수 있는 장점이 존재함
전반적으로는 당근마켓 포맷을 표방함.
